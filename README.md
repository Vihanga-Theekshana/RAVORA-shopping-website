# 🛍️ RAVORA 

RAVORA is a full-stack e-commerce web application for clothing shopping.  
It includes authentication, cart management, admin dashboard, and secure payment integration.

---

## 🚀 Features

- User authentication (JWT)
- Forgot password with OTP verification
- Product listing and details
- Add to cart / remove from cart
- Admin dashboard
- Upload multiple product images
- Secure checkout
- Online payment integration (PayHere)
- Protected routes (Admin & User)

---

## 🔐 Authentication System

- Login & Register using JWT
- Forgot password using OTP (email verification)
- Secure route protection for users and admin

---

## 💳 Payment Integration

RAVORA uses **PayHere** for online payments.

- Sandbox payment gateway for testing
- Payment redirection to PayHere
- Return URL (success / cancel)
- Notify URL for backend confirmation
